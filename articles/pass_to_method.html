    Passing the ValueStringBuilder to a method

    As ValueStringBuilder is a ref struct, you should be careful when passing the instance around. You should pass by-reference and not by-value.

    public void MyFunction()
{
    using var stringBuilder = new ValueStringBuilder();
    stringBuilder.Append("Hello ");
    AppendMore(ref stringBuilder);
}

private void AppendMore(ref ValueStringBuilder stringBuilder)
{
    stringBuilder.Append("World");
}

    This will print: Hello World

    ⚠️ If the ref keywords are omitted, the builder is passed by-value and this will print: Hello .
